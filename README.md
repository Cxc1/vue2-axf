  ## 启动项目
  
  ``` bash
 # 安装依赖
 npm install
 
 # 运行项目自： localhost:8080
 npm run dev
 
 # 打包项目
 npm run build
 
 # build for production and view the bundle analyzer report
 npm run build --report
 ```
 
 初始化`mock`数据通过： `npm run db`
